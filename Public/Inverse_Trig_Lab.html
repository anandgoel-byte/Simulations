Conceptual Assessments

Question 1

Why do we have to "restrict" the domain of the original functions?

A function must pass the Horizontal Line Test to be invertible (meaning every y-value maps to exactly one x-value). Since trigonometric waves repeat infinitely, we must cut off a single, unique "slice" of the wave—called the Principal Value Branch—to make the inverse mathematically valid.
Question 2

Why does Arctangent ($\arctan$) have horizontal asymptotes?

The original tangent function has vertical asymptotes at $x = -\pi/2$ and $x = \pi/2$ because the slope goes to infinity. When we mirror the function across $y=x$ to create the inverse, all $x$ and $y$ values swap. Therefore, vertical asymptotes become horizontal asymptotes at $y = -\pi/2$ and $y = \pi/2$.
Question 3

Can you calculate $\arcsin(2)$?

No, it is undefined! The domain of $\arcsin(x)$ is strictly $[-1, 1]$ because the range of the original $\sin(x)$ function never exceeds those bounds. There is no real angle whose sine is $2$.
