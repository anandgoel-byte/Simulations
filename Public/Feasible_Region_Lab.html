Conceptual Assessments

Question 1

What is the Corner Point Theorem?

The Fundamental Theorem of Linear Programming states that if an objective function has an optimal value (maximum or minimum) within a bounded feasible region, that optimal value will always occur at one of the vertices (corner points) of the region.
Question 2

What does the dashed Objective Line represent visually?

Often called an "Isoprofit" or "Isocost" line, it represents all combinations of $(x,y)$ that yield a specific target $Z$ value. As you drag the $Z$ slider, the line moves parallel to itself. The optimal solution is the very last point the line touches before leaving the feasible region entirely.
Question 3

What is a "Binding" or "Active" constraint?

A constraint is binding if it physically dictates the optimal solution point (i.e., the optimal point lies exactly on the line $ax+by=c$). If a constraint line bounds the region but the optimal point occurs elsewhere, it is non-binding (meaning there is "slack" or leftover capacity).
