📐 SHARED ALTITUDE GUIDE

Welcome to the Shared Altitude Tracing Laboratory! This workspace demonstrates that triangles can share heights even when skewed or having different shapes.

📐 Draggable Vertices

Drag any primary corner node (blue/purple) to modify the geometries. You can skew them to force the altitudes outside the bases (obtuse cases).

⚡ Triangle Presets

Use the **Presets** sidebar to load classic sub-triangles. Each preset will automatically select the base segment and height vertex on the canvas.

🎬 Altitude Animation tracing

Select a triangle and click **Trace Altitude**. The canvas will visually project the base line, draw base extensions if needed, drop the perpendicular height, and pulse a right-angle symbol.
