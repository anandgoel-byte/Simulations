📘 FUNCTION MAPPING PROTOCOLS

Welcome to the 3D Function Mapping Visualizer! Understand how elements map from the Domain to the Codomain.

🎯 Visual Key

  • Domain (Set A): Cyan nodes on the Left. Codomain (Set B): Pink nodes on the Right.
  • Injective (One-One): Watch the lines. If no two lines land on the exact same Pink node, it is Injective.
  • Many-One: If a Pink node receives multiple lines, the system highlights it in Yellow.
  • Surjective (Onto): If all Pink nodes are connected, the Range equals the Codomain.
  • Into: If there is an isolated, unconnected Pink node (highlighted hollow), the Range is a proper subset of the Codomain.

🎓 Learning Outcomes

  • Mapping Properties: Visually and algebraically characterize injective, surjective, and bijective mappings.
  • Invertibility Conditions: Master why a function must be bijective to support a well-defined inverse function.
  • Finite Cardinality: Explore how the sizes of finite sets A and B restrict the types of possible functions.
