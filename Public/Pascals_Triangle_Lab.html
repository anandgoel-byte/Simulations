Question 1

What does each number in Pascal's Triangle represent mathematically?

Each number represents the number of ways to choose $k$ items from a set of $n$ items, mathematically written as $nCk$ or $\binom{n}{k}$. It is the absolute core of combinatorics!
Question 2

Why do the values in row $n$ always sum exactly to $2^n$?

The sum of the $n$-th row is $\sum \binom{n}{k}$. This is equivalent to counting the total number of possible subsets of a set with $n$ elements. Each element can either be "in" or "out" (2 choices), so $n$ elements give $2 \times 2 \times \dots = 2^n$ total subsets.
Question 3

How is this related to the Binomial Theorem?

The Binomial Theorem expands $(x+y)^n$. When multiplied out, the coefficient of each $x^{n-k}y^k$ term is exactly the number of ways to pick $y$ from $k$ of the binomial factors, which is $\binom{n}{k}$ — matching the $n$-th row of Pascal's Triangle perfectly!
