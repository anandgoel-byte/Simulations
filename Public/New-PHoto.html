🔬 Guided Virtual Laboratory Tasks

Task 1

Determine Threshold Frequency of Sodium

1. Select Sodium ($Na$) as cathode (Work function $\Phi \approx 2.36\text{ eV}$).
2. Set plate voltage to $0.0\text{ V}$ and intensity to $50\%$.
3. Sweep wavelength slowly from $800\text{ nm}$ down to UV and record where Ammeter is non-zero. Verify that $f_0 = \frac{c}{\lambda_0} \approx 5.7 \times 10^{14}\text{ Hz}$.

Task 2

Measure Planck's Constant ($h$) via stopping potentials

1. Select Potassium ($K$, $\Phi \approx 2.29\text{ eV}$).
2. Set wavelength to $300\text{ nm}$, $400\text{ nm}$, $500\text{ nm}$ successively.
3. For each, adjust voltage negatively until photocurrent is exactly $0.00\ \mu\text{A}$ and record $V_{\text{stop}}$. Plot values to verify slope is $\frac{h}{e}$.
