What is a Set?

A Set is a well-defined collection of distinct objects. The objects belonging to a set are called its elements or members. "Well-defined" means that for any object, we must be able to decide immediately whether it belongs to the collection ($\in$) or not ($\notin$).

Conceptual Assessments

Question 1

How is set membership mathematically represented?

If an element $x$ belongs to a set $A$, it is written using the epsilon symbol: $$x \in A$$ If $y$ does not belong to set $A$, it is written: $$y \notin A$$
Question 2

What is the difference between Roster Form and Set-Builder Notation?

Roster form lists all elements explicitly: $A = \{1, 2, 3\}$. Set-builder notation specifies the defining properties of elements: $$A = \{ x \in \mathbb{N} \mid x < 4 \}$$
Question 3

How is an Empty Set mathematically defined?

A set containing no elements is the Empty (or Null) Set, denoted by $\emptyset$ or $\{\}$. $$\emptyset = \{ x \mid x \neq x \}$$ Its cardinality is strictly zero: $|\emptyset| = 0$.
Question 4

What is a Power Set, and how is its cardinality calculated?

The Power Set $P(A)$ is the set of all subsets of $A$. If set $A$ has $n$ elements, the power set possesses $2^n$ elements: $$|P(A)| = 2^{|A|}$$
Question 5

How is the Cartesian Product of two sets defined?

The Cartesian Product $A \times B$ is the set of all ordered pairs $(a, b)$ where $a \in A$ and $b \in B$: $$A \times B = \{ (a, b) \mid a \in A \land b \in B \}$$
