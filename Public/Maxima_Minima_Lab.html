Conceptual Assessments

Question 1

What makes a point a "Critical Point"?

A critical point occurs wherever the first derivative $f'(x) = 0$ (or is undefined). Geometrically, this means the tangent line to the curve is perfectly horizontal. These points are the only candidates for being local maxima or minima.
Question 2

How does the Second Derivative Test work?

Evaluate $f''(x)$ at a critical point. If $f''(x) > 0$, the curve is Concave Up ($\cup$), making the point a local minimum. If $f''(x) < 0$, it is Concave Down ($\cap$), making it a local maximum. If $f''(x) = 0$, the test is inconclusive.
Question 3

What happens if $A = 0$ in the cubic equation?

The function downgrades from a cubic to a quadratic (a parabola): $f(x) = Bx^2 + Cx + D$. A parabola only has one critical point (its vertex), meaning it will only ever have one global maximum OR one global minimum, but never both.
