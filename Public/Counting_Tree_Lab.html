Conceptual Assessments

Question 1

What exactly does the Fundamental Counting Principle state?

It states that if there are $m$ ways to do one thing, and $n$ ways to do another, then there are exactly $m \times n$ ways of doing both together. This scales up infinitely for any number of sequential events!
Question 2

Why use a Tree Diagram instead of just multiplying the numbers?

Multiplying gives you the total number of outcomes. A Tree Diagram actually maps out the entire Sample Space, allowing you to see and explicitly list every single specific combination (e.g., Soup $\to$ Steak $\to$ Cake).
Question 3

Does the principle change if the events are "Dependent"?

The multiplication rule still applies, but the number of choices changes! For example, if drawing 3 cards from a deck without replacement, the choices are $52 \times 51 \times 50$. The number of branches decreases at each step.
