📘 1D Kinematics Matrix — User Guide

This virtual laboratory visually matches the relationships between position, velocity, and acceleration through interactive calculus evaluation modeling.

🎯 Core Interactive Features

  • Malleable Nodes: Drag any of the 4 colorful circular handle pins inside the Velocity Arena view up or down to set unique speed profiles.
  • Boundary Constants: Adjust initial displacement properties ($x_0$) to change the position graph's initial offset.
  • Dynamic Integration Overlays: Toggle visual fill bounds to highlight the geometric accumulation of position.

📊 Mathematical Analysis Specs

  • Velocity Slope: Derivative values yield local acceleration vectors.
  • Integral Evaluation: Area tracking computes absolute change in positions over time.
