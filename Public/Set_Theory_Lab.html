📘 SET THEORY PROTOCOLS

Welcome to the Set Theory Laboratory! Visually explore how logical operators combine and isolate elements inside Venn diagrams.

🎯 Core Controls

  • Select Operation: Choose from standard operations (Intersection, Difference) or advanced Laws. The Canvas will instantly draw the required 2-Set or 3-Set diagram.
  • LHS vs RHS Toggle: When a Law is selected (like De Morgan's), flip between the Left-Hand and Right-Hand formulas. Notice how the mathematical equation changes, but the Highlighted Region and Final Result stay exactly the same! This visually proves the law.
  • Evaluate Region: Click the pink Evaluate button to run the Boolean calculation and shade the resulting geometric sector in glowing pink.

🎓 Learning Outcomes

  • Set Algebra Operations: Visualize and define Union, Intersection, Difference, Symmetric Difference, and Complements.
  • Visual Proofs of Laws: Verify algebraic laws (De Morgan's, Distributive) by comparing LHS and RHS highlighted regions.
  • Cardinality Quantification: Relate visual intersections and unions directly to quantitative element counting.
